# บทที่ 2 ติดตั้ง Git #

Git จะถูกแยกออกเป็น 2 ส่วน คือ Git SCM กับ Source Tree สำหรับคนที่ต้องการใช้งาน git ผ่านทาง command line สามารถติดตั้งเฉพาะ Git SCM อย่างเดียวก็พอ ส่วนสำหรับใครที่ถนัด GUI ก็ต้องติดตั้ง Source Tree ด้วย เพื่อให้สามารถใช้งาน Git แบบ Graphic Mode ได้ สำหรับหนังสือเล่มนี้ผมนำเสนอวิธีใช้งาน Git แบบ Graphic Mode เพราะฉะนั้นเราจะติดตั้ง Source Tree ด้วย

## เปิดใช้งาน source tree ครั้งแรก ##

เมื่อเปิดใช้งาน source tree ครั้งแรก มันจะให้เราตั้งค่าการใช้งานเบื้องต้นนิดหน่อย

1.	ติ๊กเครื่องหมายถูกหน้าช่อง I agree the..., Help improve... และ Configurate automatic... จากนั้นคลิกปุ่ม Continue
	
	![](images/chapter-2/source-tree-2014-12-15-16-53-55.png)

2.	รอสักครู่หาก Source Tree ทำการดาวน์โหลดข้อมูลบางอย่าง
	
	![](images/chapter-2/source-tree-2014-12-15-16-54-07.png)    
    
3.	คลิกปุ่ม Next
	
	![](images/chapter-2/source-tree-2014-12-15-16-55-55.png)    
    
4.	Source Tree ถามให้ติดตั้ง ignore ไฟล์ให้คลิกปุ่ม Yes ในบทข้างหน้าท่านจะได้ศึกษาเรื่อง ignore ไฟล์ ไฟล์ที่เป็นตัวกำหนดว่าไฟล์อะไรบ้างที่ Source Tree ไม่ต้องสนใจในโปรเจ็กของเรา
	
	![](images/chapter-2/source-tree-2014-12-15-16-56-06.png)    
    
5.	หน้าต่างป้อน Remote account ตอนนี้ให้ข้ามไปก่อน เราค่อยไปทำความเข้าใจและใช้งานทีหลัง คลิกปุ่ม Skip Setup
	
	![](images/chapter-2/source-tree-2014-12-15-16-56-17.png)
    
6.	หน้าต่าง Load SSH Key? เราจะข้ามการกำหนดการทำ Authentication ด้วย SSH Key ไปก่อน เราจะทำความเข้าใจเรื่องนี้อย่างละเอียดอีกครั้งในบทข้างหน้า คลิกปุ่ม No
	
	![](images/chapter-2/source-tree-2014-12-15-16-56-24.png)    
    
7.	หน้าตา Source Tree จะเป็นดังภาพ
	
	![](images/chapter-2/source-tree-2014-12-15-16-56-37.png)    
    

Source Tree เป็นโปรแกรม Git GUI เพราะฉะนั้นเมื่อเราพูดว่าเราใช้ Source Tree นั่นก็หมายความว่าเรากำลังใช้ Git 


## Configuration ##
ก่อนจะทำอย่างอื่นต่อไป สิ่งที่เราจะต้องทำเป็นสิ่งแรกก็คือ บอกความเป็นตัวเราเสียก่อน คือ ชื่อ-นามสกุล และอีเมล์ สองสิ่งนี้จำเป็นต้องใช้เพราะว่า Git มันจะบันทึกไว้ว่าคนชื่ออะไรเขียนโค้ดบรรทัดนั้นๆ และจะติดต่อเขาได้ทางไหน


1.	เปิด Source Tree ขึ้นมา คลิกเมนู Tools > Options
	
	![](images/chapter-2/source_tree_configuration_01.png)    

2.	หน้าต่าง Options จะถูกเปิดขึ้นมาป้อนข้อมูล full name, Eamil address เสร็จแล้ว คลิกปุ่ม OK 
    ในช่อง full Name: ป้อนชื่อ-นามสกุล
    ในช่อง Email address: ป้อนอีเมล์
	
	![](images/chapter-2/source_tree_configuration_02.png)
